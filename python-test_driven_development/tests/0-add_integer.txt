File: 0-add_integer.py
Function: add_integer(a, b=98)

Description:
This function adds two integers. It accepts two arguments, which can be integers or floats, and returns their sum as an integer.

Arguments:
- a: first number (int or float)
- b: second number (int or float, default = 98)

Return:
- The sum of a and b, cast to int

Exceptions:
- Raises TypeError if a or b are not integers or floats
